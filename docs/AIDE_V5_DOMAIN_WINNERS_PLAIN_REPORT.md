# AIDE v5 Domain Winners Plain-Language Report

## What This Report Covers

This report focuses only on the best composition found for each of the 42 physics domains from the AIDE v5 variation sweep. It leaves out the full project architecture and stays centered on one question: for each domain, which composition came out best, where would we use it, why did it win, and what should we be careful about?

- Source variation sweep rows: **32,760**
- Output parts: **28**
- Domains covered: **42**
- Top shortlist kept per domain: **50**
- Main support files:
  `dataset_exports/aide_domain_variation_pack/aide_domain_winners.csv`
  `dataset_exports/aide_domain_variation_pack/domain_toplists/`

## Big Picture

- The broad all-round winner was **Rene 41**, which took **19** of the 42 domains. That tells us the current engine strongly rewards high-temperature, high-stability superalloy chemistry across many domains.
- The strongest practical aluminum winner was **5083-H116**, which took **6** domains, especially manufacturing, compliance, and practical-use domains.
- **316L** and **2507** were the stainless specialists. They dominated the domains where corrosion resistance, weldability, hydrogen tolerance, and service reliability mattered most.
- Titanium compositions stayed important in human-contact and lightweight service: **Ti-3Al-2.5V** won biocompatibility, and **Beta-21S** won structural efficiency.
- A few domains should be read more cautiously than others. In particular, **CALPHAD Stability** and **Shape Memory** are still more like screening indicators than deeply calibrated final-property predictors.

## How To Read Each Entry

- `Winner composition` tells us the composition that came out on top for that domain.
- `Best use` explains where that winner is most useful in simple practical terms.
- `Why it won` comes from the strongest positive checks in that domain.
- `Watch out` highlights the main weakness, warning, or tradeoff that still showed up around the same composition.

## Final Takeaway

If we want one simple conclusion from the full 42-domain winner table, it is this: **AIDE v5 currently prefers specialist families for specialist jobs, but it rewards Rene 41-like superalloy chemistry whenever the domain values thermal stability, mechanical robustness, or oxidation-resistant service.** Meanwhile, practical aluminum, stainless, titanium, magnesium, copper, refractory, and HEA families still show up exactly where we would hope: manufacturing, corrosion, biocompatibility, hydrogen, optical, superconducting, and nuclear-specific domains.

That makes this winner table useful as a quick alloy-selection map: if we know the domain that matters most, we can immediately jump to the family that AIDE v5 currently trusts the most for that problem.
